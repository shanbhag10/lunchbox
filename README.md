# lunchbox
A web app that connects local home chefs to busy folks for healthy home-made food

## Commands 
### Kill flask process
`ps aux | grep 'python'`
`kill -9 <PID>`

### heroku python
`heroku run python -a lunchbox-app`

### heroku psql
`heroku pg:psql -a lunchbox-app`

### local psql
`psql lunchbox postgres`

### heroku open
`heroku open -a lunchbox-app`
